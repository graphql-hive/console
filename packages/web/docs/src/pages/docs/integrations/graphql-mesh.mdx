# GraphQL-Mesh

[GraphQL-Mesh](https://the-guild.dev/graphql/mesh) is a framework that helps shape and build an
executable GraphQL schema from multiple data sources.

GraphQL-Mesh supports Hive natively with a plugin.

## Installation

<PackageCmd packages={['@graphql-mesh/plugin-hive']} />

## Integration Guide

You can follow the [Hive Integration guide](https://the-guild.dev/graphql/mesh/docs/guides/hive) for
the complete instructions.
